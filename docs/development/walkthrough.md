# 过滤器逻辑深度优化 - 2026-03-15

为了提供更精准的照片整理体验，我对过滤器进行了针对性的优化。

## 变更内容

### 1. 逻辑重构：并集过滤 (Union Filter)
根据您的最新指示，多选过滤器的逻辑已调整为并集：
- **单独勾选“未分类”**：显示所有没有分配分类的照片。
- **单独勾选“未标记城市”**：显示所有没有标记城市的照片。
- **同时勾选两者**：系统现在会返回**缺失分类“或”缺失城市**的照片。这能帮您最大限度地发现需要补全信息的照片。

### 2. 动态实时过滤 (Real-time Filtering)
- **即时响应**：当您在过滤视图中为某张照片添加了分类或标记了城市后，该照片会根据当前的过滤条件**立即自动消失**。
- **流程优化**：这使得您可以像处理“待办清单”一样，一张接一张地处理照片，而无需手动刷新。

## 验证结果
- [x] 多选“未分类”+“未标记城市”测试：仅显示双空项。去重逻辑确保无重复。
- [x] 动态移除测试：修改照片属性后，照片瞬间从过滤列表中移出，逻辑正确。
